Flask-Restaction
-----------------

Flask-Restaction 是一个为 RESTful API 而生的 Web 框架

- 创建 RESTful API

- 校验用户输入以及将输出转化成合适的响应格式

- 身份验证和权限控制

- 自动生成 Javascript SDK 和 API 文档

注意：仅支持 Python3.3+

Flask-Restaction 依赖 `Validater <https://github.com/guyskk/validater>`_ 校验输入输出。


安装
~~~~~~~~~~~~~

.. code:: python

    pip install flask-restaction
